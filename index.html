About me

My name is Robert Spottiswood.
I’m a programmer-in-training and an aspiring game developer.

I’ve been fiddling around with code of various different programming languages and when I heard about a class for beginner developing I jumped at the chance to have a more active experience with computers.

I’ve tried making things in plenty of different programming languages, different software environments, different game engines, even across different operating systems.

Of course, when I’m not experimenting with technology, I’m probably playing games. I mean, who doesn’t like playing games? It’s because of all of the fantastic games out there that I’ve been so inspired to create something, anything, that’ll make a mark on the technological world and make a positive contribution to the technological world we now live in.
